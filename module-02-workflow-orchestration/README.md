## Module 02 – Workflow Orchestration (Kestra)

This module covers end-to-end workflow orchestration using Kestra, including:

- Local ETL pipelines with Postgres
- Scheduled workflows and historical backfills
- Cloud ELT pipelines on GCP (GCS + BigQuery)
- Idempotent loads using staging tables and MERGE
- AI-assisted workflow development with Kestra AI Copilot
- Retrieval Augmented Generation (RAG) examples using Gemini
